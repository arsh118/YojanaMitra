# YojanaMitra — MVP

Quickstart:
1. Copy `.env.local.example` → `.env.local` and fill keys.
2. Install deps: `npm install`
3. Seed sample schemes: `npm run seed`
4. Start dev server: `npm run dev`
5. Open http://localhost:3000

This repository contains:
- simple chat UI at `/`
- Whisper transcription API at `/api/whisper`
- Profile extraction using OpenAI at `/api/profile`
- RAG & deterministic match at `/api/match`
- PDF autofill sample at `/api/fillpdf`
